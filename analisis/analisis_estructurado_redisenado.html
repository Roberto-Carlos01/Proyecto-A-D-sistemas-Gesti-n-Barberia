Diccionario de Datos

II.4. Diccionario de datos

Tabla: USUARIO

Campo Tipo de Dato Longitud Nulo Clave Descripción
idUsuario INT 11 NO PK Identificador único del usuario
nombreUsuario VARCHAR 20 NO Nombre de usuario para inicio de sesión
password VARCHAR 30 NO Contraseña cifrada del usuario
correo Electronico VARCHAR 30 NO Correo electrónico asociado a la cuenta

Tabla: EMPLEADO

Campo Tipo de Dato Longitud Nulo Clave Descripción
idEmpleado INT 11 NO PK Identificador único del empleado
idUsuario INT 11 NO FK Referencia al usuario del sistema asociado
nombre VARCHAR 40 NO Nombre completo del empleado
ci VARCHAR 20 NO Cédula de identidad del empleado
teléfono VARCHAR 20 NO Número de contacto del empleado
dirección VARCHAR 100 Si Dirección domiciliaria del empleado
fechaNacimiento DATE NO Fecha de nacimiento del empleado
foto VARCHAR 255 Si Ruta o URL de la foto de perfil
horario VARCHAR 100 NO Horario de trabajo asignado
estado VARCHAR 20 NO Estado laboral (activo / inactivo)
rol VARCHAR 50 NO Rol dentro de la barberia (barbero, administrador, etc.)

Tabla: BARBERO

Campo Tipo de Dato Longitud Nulo Clave Descripción
idBarbero INT 11 NO PK Identificador único del barbero
idEmpleado INT 11 NO FK Referencia al registro de empleado base
especialidad VARCHAR 40 Sí Área de especialización del barbero
comision Base DECIMAL 10.2 SÍ Porcentaje o monto base de comisión por servicio
idSucursal INT 11 NO FK Identificador único de la sucursal

Tabla: CLIENTE

Campo Tipo de Dato Longitud Nulo Clave Descripción
idCliente INT 11 NO PK Identificador único del cliente
idUsuario INT 11 NO FK Referencia al usuario del sistema (si tiene cuenta)
nombre VARCHAR 100 NO Nombre completo del cliente
teléfono VARCHAR 20 SÍ Número de contacto del cliente
ci VARCHAR 20 Si Cédula de identidad del cliente
fechaNacimiento DATE Si Fecha de nacimiento del cliente
oficio VARCHAR 100 Si Ocupación o profesión del cliente
puntosGanados INT 11 NO Puntos acumulados por el cliente en el programa de fidelidad

Tabla: SERVICIO

Campo Tipo de Dato Longitud Nulo Clave Descripción
idServicio INT 11 NO PK Identificador único del servicio
nombre VARCHAR 100 NO Nombre del servicio ofrecido
descripción TEXT Si Descripción detallada del servicio
precio DECIMAL 10,2 NO Precio estándar del servicio
duracionAproximada INT SÍ Duración estimada del servicio en minutos
comisionBarbero DECIMAL 0.50 NO 50% de comisión para el barbero al realizar el servicio

Tabla: CITA/RESERVA

Campo Tipo de Dato Longitud Nulo Clave Descripción
idReserva INT 11 NO PK Identificador único de la cita / reserva
idCliente INT 11 NO FK Cliente que realiza la cita
idBarbero INT 11 NO FK Barbero asignado para atender la cita
fecha DATE NO Fecha programada de la cita
hora TIME NO Hora programada de la cita
estado VARCHAR 30 NO Estado de la cita (pendiente, confirmada, cancelada, completada)
detalles TEXT Sí Observaciones o detalles adicionales de la cita
monto TotalAdelantado DECIMAL 10,2 SI Monto adelantado o al momento de reservar

Tabla: ATENCION

Campo Tipo de Dato Longitud Nulo Clave Descripción
idAtencion INT 11 NO PK Identificador único del registro de atención
idCita INT 11 Sí FK Cita de origen de esta atención
idBarbero INT 11 NO FK Empleado que realizó la atención
fecha DATE NO Fecha en que se realizó la atención
hora TIME NO Hora de inicio de la atención
monto Total DECIMAL 10,2 NO Monto total cobrado por la atención
valoración INT SI Calificación del cliente sobre la atención (escala 1-5)
idCliente INT 11 NO FK Cliente recibe la atención

Tabla: INVENTARIO

Campo Tipo de Dato Longitud Nulo Clave Descripción
idInventario INT 11 NO PK Identificador único del registro de inventario
idProducto INT 11 NO FK Producto al que corresponde el inventario
idSucursal INT 11 Sí FK Sucursal donde se ubica el stock
stockActual INT 11 NO Cantidad actual disponible en stock
stockMinimo INT 11 Si Cantidad mínima permitida antes de alertar reposición

* El resto de tablas de intersección (ASISTENCIA, CAPACITACION, OFERTA, PROMOCION, PRODUCTOS) siguen la misma normalización relacional detallada en el documento oficial.
